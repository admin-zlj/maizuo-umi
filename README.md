# maizuo

## 项目简介

模拟 maizuo电影

技术栈：umi+ antd-mobile + json-server

## 项目启动

在json-server_mock终端

```powershell
json-server db.json --watch
```

在maizuo文件夹终端

```js
npm i
```

```js
npm start 
```
